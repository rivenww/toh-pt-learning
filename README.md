### Prerequisites

- Install [Node.js] which includes [Node Package Manager][npm]

### Setting Up a Project

```
npm install
```

Run the application:

```
cd toh-pt-learning
ng serve
```

If you wanna run the application in proxy mode:

```
cd toh-pt-learning
npm run startProxy
```
After launch the backend project, you could get data from DynamoDB.
